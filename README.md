# 🍳 Digital Recipe Management App

A simple yet functional **Python-based Command-Line Interface (CLI) application** that allows users to manage recipes efficiently. Users can add, view, search, and delete recipes using an interactive menu system.

This project demonstrates modular programming, data structures, input validation, and exception handling in Python.

---

## 📌 Features

### ➕ Add Recipe
- Input recipe name and type
- Add multiple ingredients
- Add step-by-step cooking instructions
- Automatically stores the recipe in memory

### 📖 View Recipes
- Displays all saved recipes
- Shows recipe name, type, ingredients, and steps
- Clean formatted output for readability

### 🔍 Search Recipe
- Search by recipe name (case-insensitive)
- Displays complete recipe details if found

### 🗑 Delete Recipe
- Remove a recipe by name
- Confirms successful deletion
- Handles missing recipes properly

### 🚪 Exit Program
- Safely exits the application
